# Presentation3 Project

This project is a Beamer presentation created for the purpose of showcasing the findings and methodologies related to the mouse maze project.

## Project Structure

- **powerpoint/**: Contains all files related to the Beamer presentation.
  - **pres3.tex**: The main LaTeX document for the presentation.
  - **sections/**: Contains individual section files for the presentation.
    - **introduction.tex**: Content for the introduction slide.
    - **methodology.tex**: Content for the methodology section.
    - **results.tex**: Content for the results section.
    - **conclusion.tex**: Content for the conclusion section.
  - **images/**: Directory for images used in the presentation (currently contains a .gitkeep file).

- **written_report/**: This directory contains the written report related to the project.

## Usage

To compile the presentation, navigate to the `powerpoint` directory and run the LaTeX compiler on `pres3.tex`. Ensure that all required packages for Beamer are installed.

## Contribution

Feel free to contribute by adding new sections or images to enhance the presentation. Make sure to follow the existing structure for consistency.

\begin{frame}
\frametitle{Issues Remaining}
\begin{itemize}
    \item Get a deeper understanding of ROS2
    \item Improve understanding of pathfinding algorithms
    \item Get a deeper understanding of how the ROS2 Control Flow works
    \item Implement more pathfinding algorithms in Rust
    \item Improve robot control in Rust simulation
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Lessons Learned}
\begin{itemize}
    \item ROS2 simulates motors, sensors, physics, and control flow
    \item Rust is good if you want to simulate something such as pathfinding algorithms
    \item Bevy is a good ECS framework for simulations
    \item ROS2 on Mac is difficult
    \item Python is good for data analysis and visualization
    \item What is the problem you want to focus on?
    \item Algorithms, sensors, control - just a small piece in a big system?
    \item Solving the bigger problem and offloading the details to existing tools is sometimes better
    \item Depends on your goals
\end{itemize}
\end{frame}
